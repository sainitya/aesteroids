# Asteroids Game

This project is a retro-style Asteroids clone written in Python using SimpleGUI and SimpleGUICS2Pygame. It was originally created as a college mini-project and later refined into a playable version of the classic arcade game.

## What the game does

- Control a spaceship and fly around the screen
- Shoot asteroids to earn points
- Avoid crashing into rocks or getting hit by debris
- Use bombs occasionally to clear the screen
- Enjoy animated explosions and background effects

## Files

- astero.py - main game implementation
- space ship prototype.py - a prototype or draft version of the ship logic
- user33_Bhc7VzXKbXGVQV1.py, user38_*.py, and user40_*.py - supporting helper files used by the game

## How to run

This game is intended to run with Python and the SimpleGUICS2Pygame library.

1. Install the dependency:
   ```bash
   pip install SimpleGUICS2Pygame
   ```
2. Run the main game file:
   ```bash
   python astero.py
   ```

## Notes

- The game was originally written years ago and may run better in older browser-based environments.
- There is a known issue with asteroid-splitting collisions that may need fixing later.

## Summary

This is a nostalgic Asteroids-style game built as a learning project, inspired by classic arcade gameplay and implemented in Python.
